# Excel-Technical-Project
Excel Technical Project
Financial Income Statement Forecasting Model

## Project Overview
This project is an automated, macro-enabled Excel financial model designed to forecast an insurance company's simplified Income Statement over a 4-year period (2022–2025) based on historical performance from 2021. I finished this project as a way to practice Excel: implementing vba macro loops, data validation, and index/match formulas.

* **Scenario Analysis:** Allows real-time toggling between Bull, Base, and Bear cases to evaluate sensitivity to market conditions, along with percentage changes to base assumptions.

* **Excel VBA Automation:** Features embedded VBA routines for scenario execution in order to map out scenarios in which future interest rates change to 2%, 4%, 6%, or 8%.
